File System Manager

Features

  • Create, Read, Update, and Delete files (CRUD).
  • Manage directories and subdirectories.
  • Search for files with advanced filters.
  • View file properties such as size, creation date, and type.
  • Simulate virtual memory usage for file management.
  • Integrated caching for frequently accessed files.
  • Multithreaded operations for faster processing.
  • Interactive graphical interface for file system visualization.
